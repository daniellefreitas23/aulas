Id e Class

ID

O ID funciona como um CPF. Assim como só pode ter um número de CPF, cada elemento só pode ter um valor de ID. O CPF é único, ou seja, ou seja ningém pode ter um CPF igual o seu. Isso significa que o ID tambémé único, então não é possível repetir o mesmo valor duas vezes

CLASS

O CLASS funciona de maneira oposta ao ID. Diferente do ID em que você só pode ter um valor, no CLASS você pode ter vários valores. Também diferente do ID, em que não se pode repetir um valor, no CLASS você pode utilizar o mesmo valor para vários elementos.
